# 三峡库区库水位采集脚本(V1.00 Dorad,blog.cuger.cn)
## 介绍
本脚本用于采集三峡库区每日库水位数据，数据来源为[中国长江三峡集团有限公司](!http://www.ctg.com.cn/sxjt/sqqk/index.html)
采集结果放置在 mysql 数据库中, 方便用户查询下载。
## 使用说明
此脚本用于服务端。自动自动采集数据，并存到数据库，提供文件下载。
